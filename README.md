## Sales Performance & Returns Analysis Dashboard

## Project Overview

This project presents an interactive Tableau dashboard developed using Microsoft Excel data to analyze sales performance, profitability, order volume, and product return trends through visual analytics and KPI-driven reporting.

## Business Objective

- Analyze sales, profit, and order performance across different time periods.
- Monitor monthly business trends and category performance.
- Identify sub-categories with high return rates.
- Support data-driven business decision-making.

---

## Dashboard Components

### KPI Overview
- Total Sales
- Total Profit
- Total Orders
- Return Percentage

### Sales & Profit Trend Analysis
- Dual-axis line chart comparing monthly sales and profit trends.
- Trend analysis across different time periods.

### Category Performance Analysis
- Sales and profit comparison across product categories.

### Return Rate Analysis
- Top sub-categories ranked by return percentage.

### Interactive Filtering
- Year-wise performance analysis using dashboard filters.
